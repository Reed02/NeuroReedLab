Script 04 Methods

Sliding Window Dominant Frequency

One second window, 100ms step. Tracks dominant frequency in 8 to 80 Hz over time per channel.

MATLAB Time Frequency
04 sliding window frequency.m
Motivation

The FFT scripts so far compute a static average over the entire segment. But motor imagery is dynamic. The brain oscillatory state changes as the person imagines a movement and then returns to rest. A sliding window analysis reveals how dominant frequency evolves over the segment duration, which is far more informative than a single spectral snapshot.

Setup & Usage

This script reads .set files, so the dataset to use here is EEG Classification SetFormat.zip on Zenodo. This contains the organized segment classes, but not the Eyes Open or Eyes Closed baselines. Those were originally .edf recordings that never needed segmenting, so they were never converted into .set format alongside the segments.

To run this on a specific segment, add the full EEG_Classification folder and all its subfolders to your MATLAB path, then navigate into the class folder containing the file you want before running the script. For example, to look at S001R03_segment_004_T1.set, enter the Task1_Real_Left_Fist folder in MATLAB and run 04_sliding_window_frequency.m from there.

Core Code
04 sliding window frequency.m MATLAB
window_length_samples = window_length_sec * fs; step_size_samples = step_size_sec * fs; for ch = 1:num_channels for i = 1:num_windows start_idx = (i-1)*step_size_samples + 1; segment = data(ch, start_idx:start_idx+window_length_samples-1); % FFT, restrict to 8 to 80 Hz, find peak frequency valid_idx = find(f >= freq_low & f <= freq_high); [~, max_rel_idx] = max(P1(valid_idx)); max_freqs_all(ch, i) = f(valid_idx(max_rel_idx)); end end
Thought Process

The 8 to 80 Hz band focuses on beta and gamma oscillations associated with motor processing, excluding the resting alpha which would otherwise always win the peak competition. The one second window gives frequency resolution of 1 Hz per bin, sufficient to distinguish EEG bands. The 100ms step gives temporal resolution fine enough to track rapid oscillatory changes.

Plotting all 64 channels with vertical offsets allows visual comparison of how different scalp regions evolve in parallel. This revealed that frontal and motor channels show different dynamics than occipital channels during motor tasks.

What this revealed

The plots showed that dominant frequency is not static within a segment. Transitions visible around the midpoint of longer segments are consistent with subjects seeing the target appear, performing the task, and then returning to rest. Future CNN features might benefit from capturing these temporal dynamics rather than averaging over the whole segment.
