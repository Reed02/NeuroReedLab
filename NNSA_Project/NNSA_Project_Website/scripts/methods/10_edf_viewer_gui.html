EDF Viewer GUI

Full Tkinter desktop application: EDF import, channel browser, and embedded Python REPL.

PythonTkinterGUI
02_edf_viewer_gui.py
Motivation

The MATLAB scripts required reopening and editing the code to change which file was loaded or which analysis was run. A reusable interactive tool that could load any EDF, display it, and let you write analysis code against it interactively would be far faster for exploration. This script builds that tool using Python's Tkinter framework.

Architecture

Three main components. The EDF import system uses pyedflib to load one or more files, extracting channel labels, signal arrays, and time vectors into a shared db dictionary indexed by channel name. The channel browser is a split-pane view with a channel listbox and a table showing time/voltage pairs for the selected channel. The Python REPL is an embedded text editor that executes arbitrary code with full access to db, numpy, matplotlib, and scipy.

Thought Process

The db dictionary pattern — db['ChannelName'] = {'time': ..., 'signal': ...} — makes channel access in the REPL natural. Rather than remembering array indices, you type the channel name you saw in the browser. The Tools menu preloads complete runnable snippets for the most common operations so you don't need to write boilerplate from scratch during an exploratory session.

The REPL captures stdout and stderr and displays them in the terminal widget at the bottom of the window, so print statements and error messages appear inline without a separate console.
