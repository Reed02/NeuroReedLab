Channel Position Mapper

Drag-and-drop GUI: place 64 channel labels on a head outline and save positions to CSV.

Why Channel Positions Were Needed

Topographic brain maps require knowing where each electrode sits on the scalp. MNE has standard layouts built in, but the BCI2000 channel names don't perfectly match MNE's naming conventions. Rather than writing a string normalization script, a GUI was built to place positions visually using the Sharbrough 64-channel 10-20 reference diagram as a guide.

Thought Process

The normalized coordinate system (0,0) bottom-left to (1,1) top-right with head centered at (0.5, 0.5) radius 0.45 scales correctly regardless of figure size. The saved 03_channel_positions.csv has three columns: Channel, X, Y.

Drag-and-drop is implemented via matplotlib's motion_notify_event. When you click a channel name in the listbox then click on the head, the label appears and can be repositioned before the next channel is placed. The resulting CSV is committed to the repository so topomaps are fully reproducible without re-running the GUI.
