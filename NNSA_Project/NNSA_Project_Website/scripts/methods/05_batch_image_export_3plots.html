Script 05 Methods

Batch Image Export 3 Plots

First batch pipeline saves original, filtered, and smoothed PNGs for every .set file in a folder.

MATLAB Batch
05 batch image export 3 plots.m
Goal

After validating preprocessing in scripts 01 to 04 on single files, the next step was to generate plots systematically across an entire class folder. The goal was a visual dataset that could be scanned quickly to check whether movement classes look distinct in the time domain and to spot any anomalous files before they reach the CNN.

Setup & Usage

This script reads .set files not .edf so it is still recommended to run it on folders within EEG Classification SetFormat.zip on Zenodo. Unlike the single file scripts above this one batches over an entire folder rather than one file so once you run the script your files will show up and you can select your desired folder.

Thought Process

The Visible off option is essential for batch processing. Without it MATLAB opens and closes a visible window for each of hundreds of files which is slow and can interfere with other processes. Headless figures render in the background entirely.

The smoothing window was reduced from 100ms to 50ms relative to script 01 based on visual inspection. 100ms over smoothed some of the transient features in shorter segments. 50ms preserved more of the signal structure while still reducing noise.
