Script 07 — Methods

Batch Image Export — 7 Plots

Scales the 7-plot analysis to an entire folder, all plots saved into organized subfolders.

MATLABBatch
07_batch_image_export_7plots.m
What This Does

Script 06 proved the 7-plot pipeline was informative on a single file. Script 07 applies it to every .set file in a selected folder, saving 7 PNGs per file into 7 organized output subfolders. Running this on the Rest class folder and a motor task folder side by side was the first systematic visual comparison between classes.

Thought Process

Using uigetdir rather than a hardcoded path makes the script reusable across any class folder without editing the code. The folder selection dialog appears once at the start; the rest runs automatically.

The visual comparison between Rest and Task1_Real_Left_Fist FFT plots was informative: rest segments showed consistently higher power at lower frequencies, while motor task segments showed more power in the 14–30 Hz beta band. This was the first concrete visual evidence that the data contained class-discriminating information.
