Script 02 — Methods

FFT and Peak Frequency Per Channel

High-pass at 8 Hz then compute FFT per channel. Track the global peak frequency across all 64.

MATLABFFT
02_fft_peak_frequency.m
Goal

Having seen the time-domain signal, the next question is where the energy lives in frequency. Motor imagery has known frequency-domain signatures: event-related desynchronization (ERD) in the alpha and beta bands contralateral to the imagined movement. This script computes the amplitude spectrum for each channel and identifies the peak frequency across all channels, giving the dominant oscillation during the task segment.

Core Code
02_fft_peak_frequency.mMATLAB
% High-pass at 8 Hz — suppress resting alpha/delta [b,a] = butter(4, cutoff/(Fs/2), 'high'); for ch = 1:numChannels filteredData(ch,:) = filtfilt(b, a, double(data(ch,:))); end % Single-sided amplitude spectrum Y = fft(channelData); P2 = abs(Y/N); P1 = P2(1:floor(N/2)+1); P1(2:end-1) = 2*P1(2:end-1); % double non-DC terms % Track global peak across all channels [chMaxAmp, idxMax] = max(P1); if chMaxAmp > globalMaxAmp globalMaxAmp = chMaxAmp; globalMaxFreq = f(idxMax); globalMaxChannel = ch; end
Thought Process

The 8 Hz high-pass cutoff was chosen to suppress the resting alpha band which dominates baseline recordings. For motor task segments, activity above 8 Hz — particularly beta (13–30 Hz) and lower gamma — is more diagnostic. Removing sub-8 Hz components prevents the strong resting alpha from dominating the spectral peaks.

Doubling the non-DC amplitude terms gives the correct single-sided spectrum. Without this, amplitudes are half their true value because energy is split between positive and negative frequency components. The resulting values are in the original signal units (µV).

What the peak tracking showed

Peak frequencies clustered around 15–25 Hz (beta band) for task segments, consistent with known motor cortex signatures. Rest segments showed peaks closer to 8–12 Hz (alpha band). This was an early positive sign that the data contained class-discriminating frequency information.
