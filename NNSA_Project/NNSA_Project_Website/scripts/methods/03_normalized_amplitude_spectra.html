Script 03 Methods

Normalized Amplitude Spectra

Each channel normalized to max 1 then stacked with fixed vertical offsets for cross channel comparison.

MATLAB Visualization
03 normalized amplitude spectra.m
The Readability Problem

When all 64 channels amplitude spectra are plotted without normalization, channels with high amplitude dominate and low amplitude channels are invisible. The solution is to normalize each channel to a maximum of 1 and add a fixed vertical offset proportional to the channel index, creating a waterfall style plot where all 64 channels are legible simultaneously.

Setup & Usage

This script reads .set files, so the dataset to use here is EEG Classification SetFormat.zip on Zenodo. This contains the organized segment classes, but not the Eyes Open or Eyes Closed baselines. Those were originally .edf recordings that never needed segmenting, so they were never converted into .set format alongside the segments.

To run this on a specific segment, add the full EEG_Classification folder and all its subfolders to your MATLAB path, then navigate into the class folder containing the file you want before running the script. For example, to look at S001R03_segment_002_T2.set, enter the Task1_Real_Right_Fist folder in MATLAB and run 03_normalized_amplitude_spectra.m from there.

Core Code
03 normalized amplitude spectra.m MATLAB
offsetStep = 0.1; for ch = 1:numChannels % compute FFT P1 as before % normalize each channel to its own maximum P1_norm = P1 / max(P1); % offset by channel index offset = (ch - 1) * offsetStep; plot(f, P1_norm + offset); end % y axis labels at offset positions yticks(0:offsetStep:(numChannels-1)*offsetStep);
Thought Process

The 0.1 offset step was chosen empirically. With 64 channels a step of 0.1 produces a total span of 6.3 units. The normalized spectra peak at 1.0 so there is minimal overlap between adjacent channels while still fitting the entire stack on a readable figure.

Normalizing per channel is correct here because the goal is to compare frequency shape, not amplitude. Whether a channel has peak amplitude 1 microvolt or 50 microvolts is irrelevant for identifying which channels show beta activity during the task.
