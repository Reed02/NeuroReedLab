Script 08 — Methods

Extract Average FFT Features

Shift from visualization to machine learning: saves average FFT across 64 channels as compact .mat files.

MATLAB Feature Extraction
08_extract_avg_fft_features.m
The Shift from Visualization to Machine Learning

Scripts 01 through 07 produced visualizations for human inspection. This script is the pivot to machine learning: instead of saving images, it saves compact numerical feature vectors that a CNN can learn from. The feature chosen is the average FFT amplitude spectrum across all 64 channels, computed after 14 to 50 Hz bandpass filtering and 100 ms smoothing.

Each output .mat file contains two variables: avg fft row (the mean amplitude at each frequency bin) and f (the frequency axis). The EEG CNN Features Avg FFT Zenodo dataset is the direct output.

Setup & Usage

BioSig must be installed first. Download Biosig 3.8.4.zip from the MATLAB folder, unzip it, and run biosig installer.m to add it to your MATLAB path before running this script.

This script reads .edf files, so it is recommended to run it on files from EEG CNN Dataset EDF.zip on Zenodo. Aside from the one time BioSig install, there is no need to add the dataset itself to your path since the script prompts you to select a folder directly from your files at runtime.

Core Code
08_extract_avg_fft_features.m MATLAB
% Compute FFT for all channels, then average fft_all = zeros(nChannels, length(f)); for ch = 1:nChannels Y = fft(smoothed_data(ch,:)); P2 = abs(Y/N); P1 = P2(1:floor(N/2)+1); P1(2:end-1) = 2*P1(2:end-1); fft_all(ch,:) = P1; end avg_fft = mean(fft_all, 1); avg_fft_row = avg_fft(:)'; % ensure row vector save(saveFile, 'avg_fft_row', 'f');
Thought Process

Averaging across all 64 channels is a simplification chosen to keep feature dimensionality manageable for the first CNN experiments. It discards spatial information about which brain regions show elevated activity. Saving both avg fft row and f in the same file makes the feature vector self contained so each entry directly maps to its frequency axis.

Planned improvement

Averaging across channels discards spatial structure. A straightforward improvement would be to save the full 64 by N FFT matrix instead, allowing the CNN to learn which channels are most discriminative per class. This is a natural next step.
