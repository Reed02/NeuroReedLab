Artifact Detection

Automated quality control for EEG baseline recordings using statistical deviation from smoothed signal envelopes.

Python MNE Signal Quality
09_artifact_detection.py
Why Python

This stage transitions into Python using MNE, which provides stronger tools for EEG file handling and signal processing at scale. It enables automated preprocessing across large EDF datasets with consistent filtering and statistical analysis.

Setup & Usage

This script processes .edf recordings from the EEG CNN dataset on Zenodo. No MATLAB dependencies are required. The only requirement is updating the local file path to point to the dataset directory.

raw = mne.io.read_raw_edf(f, preload=True, verbose=False) raw.filter(1., 40., fir_design='firwin', verbose=False) data, times = raw[ch] data = data[0] * 1e6 smoothed = gaussian_filter1d(data, sigma=10) norm = (smoothed - np.mean(smoothed)) / np.std(smoothed) ma = np.convolve(norm, np.ones(SMOOTH_WINDOW)/SMOOTH_WINDOW, 'same') artifact_mask = np.abs(ma) > THRESHOLD_FACTOR * np.std(ma) num_artifacts = np.sum(np.diff(artifact_mask.astype(int)) == 1)
Method

The pipeline combines Gaussian smoothing with a large moving-average envelope to isolate slow baseline behavior. Artifacts are defined as excursions exceeding three standard deviations from this envelope, capturing abrupt non-neural events such as movement or electrode noise.

Outcome

Baseline recordings show variable artifact rates across subjects, but no consistent difference between eyes-open and eyes-closed conditions. This suggests that noise is primarily subject-specific rather than condition-dependent.
