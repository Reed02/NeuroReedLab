Artifact Detection

MNE-based quality check across baseline files. Flags signal excursions beyond 3 standard deviations.

PythonMNEVisualization
01_artifact_detection.py
Why Switch to Python

At this point the pipeline moved from MATLAB to Python. MNE is Python's dominant EEG analysis library and has better support for Welch PSD estimation, loading EDF metadata, and working programmatically with large file collections. This first Python script is a straightforward quality check on the baseline recordings before committing to the Python analysis pipeline.

# Load with MNE, filter 1-40 Hz raw = mne.io.read_raw_edf(f, preload=True, verbose=False) raw.filter(1., 40., fir_design='firwin', verbose=False) # Extract channel, convert to µV, Gaussian smooth data, times = raw[ch] data = data[0] * 1e6 smoothed = gaussian_filter1d(data, sigma=10) # Normalize then large moving average envelope norm = (smoothed - np.mean(smoothed)) / np.std(smoothed) ma = np.convolve(norm, np.ones(SMOOTH_WINDOW)/SMOOTH_WINDOW, 'same') # Flag excursions beyond 3 std as artifacts artifact_mask = np.abs(ma) > THRESHOLD_FACTOR * np.std(ma) num_artifacts = np.sum(np.diff(artifact_mask.astype(int)) == 1)
Thought Process

The two-stage smoothing approach — Gaussian smoothing to clean the raw signal, then a large moving average to build a slow envelope — robustly handles signals with widely varying baseline amplitudes. Artifacts are points where the normalized signal exceeds 3 standard deviations of this smooth envelope.

The OVERLAP flag switches between stacked and overlapping display modes. Stacked is better for comparing individual subjects; overlapping gives a better overall picture of the artifact rate distribution across all files at once.

This confirmed that baseline recordings have varying artifact levels across subjects, and that eyes-open and eyes-closed are similarly noisy — neither type is systematically worse.
