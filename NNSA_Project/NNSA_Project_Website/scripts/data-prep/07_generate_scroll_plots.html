Script 07 — Data Prep

Generate EEGLAB Scroll Plots

Visual quality check: renders and saves a scroll view PNG for every file in every class folder.

MATLAB EEGLAB Visualization
07_generate_scroll_plots.m
Purpose

After building a large automated pipeline it is easy to have subtle errors such as files routed to the wrong class, truncated segments, channels with flat or clipped signals that produce no errors but generate bad data. The only reliable way to catch these is visual inspection. This script uses EEGLAB’s pop_eegplot to render the standard EEG scroll view for every file across all task folders and saves each as a high resolution PNG.

Core Code
07_generate_scroll_plots.m MATLAB
% Start EEGLAB cleanly [ALLEEG, EEG, CURRENTSET, ALLCOM] = eeglab; % Base image save location desktopPath = fullfile(getenv('HOME'), 'Desktop'); dstBase = fullfile(desktopPath, 'images'); % Folder names containing .edf files (inside current directory) taskFolders = { 'Baseline_Eyes_Open', 'Baseline_Eyes_Closed', 'Rest', 'Task1_Real_Left_Fist', 'Task1_Real_Right_Fist', 'Task3_Real_Both_Fists', 'Task4_Imag_Both_Feet', 'Task4_Imag_Both_Fists', 'Task2_Imag_Left_Fist', 'Task2_Imag_Right_Fist', 'Task3_Real_Both_Feet' }; for i = 1:length(taskFolders) srcFolder = taskFolders{i}; dstFolder = fullfile(dstBase, srcFolder); if ~exist(dstFolder, 'dir') mkdir(dstFolder); end % Find all EDF files edfFiles = dir(fullfile(srcFolder, '*.edf')); fprintf('\nProcessing folder: %s\n', srcFolder); fprintf('Found %d EDF files\n', length(edfFiles)); for j = 1:length(edfFiles) edfFile = edfFiles(j).name; [~, baseName, ~] = fileparts(edfFile); try fprintf('Loading: %s\n', edfFile); % Load EDF EEG = pop_biosig(fullfile(srcFolder, edfFile)); EEG.setname = baseName; [ALLEEG, EEG, CURRENTSET] = eeg_store(ALLEEG, EEG, 0); % Close any existing eegplot windows close(findall(0, 'Type', 'figure')); % Generate EEG plot pop_eegplot(EEG, 1, 1, 1); pause(2); figs = findall(0, 'Type', 'figure'); if isempty(figs) warning('No figure found for %s', edfFile); continue; end hFig = figs(1); set(hFig, 'Color', 'white'); ax = findall(hFig, 'Type', 'axes'); for k = 1:length(ax) set(ax(k), 'Color', 'white'); end drawnow; outFile = fullfile(dstFolder, [baseName '.png']); exportgraphics(hFig, outFile, ... 'Resolution', 300, ... 'BackgroundColor', 'white' ); fprintf('Saved image: %s\n', outFile); close(hFig); catch ME fprintf('\nError processing %s\n', edfFile); fprintf('%s\n\n', ME.message); end end end disp('All .edf plots saved.');
Setup and Usage

BioSig must be installed (run biosig_installer.m from the unzipped Biosig3.8.4.zip, in the matlab folder) before running this script.

This script must be run from inside the folder that directly contains all of your organized class folders. For example, to generate scroll plots across every class including Eyes Open and Eyes Closed baselines, download EEG_CNN_Dataset_EDF.zip from Zenodo, unzip it, cd into it, and then run this script.

Thought Process

The pause(2) call is necessary because pop_eegplot renders an interactive EEGLAB GUI figure asynchronously. Without a pause, exportgraphics can capture a blank or partially rendered window. Two seconds was sufficient on the development machine; slower systems may need more time.

The script forces figure and axes backgrounds to white before export because EEGLAB GUI themes can otherwise produce gray or transparent backgrounds that reduce readability. Wrapping each file in a try catch ensures that corrupted or unreadable EDF files do not halt the entire batch process.

This script processes EDF files produced by the previous conversion step and saves images into a mirrored folder structure under an images directory for quick visual inspection.

Manual Inspection Strategy

The goal is not to inspect every image in detail but to quickly scan for anomalies such as flat channels, clipping, or misrouted files. Most plots should look normal. Any that stand out significantly should be flagged for review.
