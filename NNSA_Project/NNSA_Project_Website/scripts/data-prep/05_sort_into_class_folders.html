Script 05 — Data Prep

Sort into Classification Folders

Copies verified segments into EEG_Classification with 9 named class subfolders.

MATLAB
05_sort_into_class_folders.m
What This Does

With counts verified, this script does the actual copying. It uses the same run-number-to-task mapping and loops through all segment subfolders, using copyfile (not movefile) to copy each .set and .fdt file into the appropriate class subfolder inside EEG_Classification/. Copying rather than moving keeps EEG_Segments/ intact as a backup.

The result is 9 folders: Rest, Task1_Real_Left_Fist, Task1_Real_Right_Fist, Task2_Imag_Left_Fist, Task2_Imag_Right_Fist, Task3_Real_Both_Fists, Task3_Real_Both_Feet, Task4_Imag_Both_Fists, Task4_Imag_Both_Feet. The EEG_Classification_SetFormat Zenodo dataset is the direct output.

Setup & Usage

No new dependencies. Once 03_segment_all_subjects.m has produced the EEG_Segments/ folder, this script can be run directly — it will create EEG_Classification/ alongside it with the 9 organized class subfolders.

Routing Logic
05_sort_into_class_folders.mMATLAB
if contains(fname, 'T0') destCategory = 'Rest'; else task = task_map(runNum); % e.g. Task1_Real switch task case 'Task1_Real' if contains(fname,'T1'), destCategory = 'Task1_Real_Left_Fist'; elseif contains(fname,'T2'), destCategory = 'Task1_Real_Right_Fist'; end % ... same pattern for Task2, Task3, Task4 ... end end copyfile(fullpath, fullfile(masterOutputFolder, destCategory));
Thought Process

The T0 shortcut is handled first: regardless of which run a rest segment comes from, it always belongs in the Rest category. This keeps the switch statement clean and avoids having a T0 arm duplicated inside every case.

Both .set and .fdt files must be copied together. EEGLAB stores large data arrays in the binary .fdt companion file; copying only the .set header produces broken files that load with no data. The file search uses [dir(*T0.set); dir(*T1.set); dir(*T2.set)] to catch all three event types.

Baseline runs are not handled here

This script only organizes the segmented motor task files. It does not sort the baseline Eyes Open (R01) or Eyes Closed (R02) recordings, since those runs are never segmented in the first place — each is a single continuous file per subject. With 109 subjects, these were originally sorted into their own folders by hand, which was tedious but manageable at that scale. It wasn't until the segmented files ballooned into the thousands that a script became necessary for organizing those.

The full EEG_Classification folder — with the segmented task classes from this script plus the hand-sorted Eyes Open and Eyes Closed baselines — is available as a ready-made download: EEG_CNN_Dataset_EDF.zip on Zenodo, linked on the Datasets page.
