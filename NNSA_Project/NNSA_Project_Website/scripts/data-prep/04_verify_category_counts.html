Script 04 — Data Prep

Category Count Verification

Dry run: counts how many segments would land in each class before moving a single file.

MATLABVerification
04_verify_category_counts.m
Why This Step Exists

The entire classification pipeline depends on a critical mapping: the run number in each filename tells you which task the recording belongs to. Runs 3, 7, and 11 are Task 1 (real fist movements); runs 4, 8, and 12 are Task 2 (imagined fist movements); and so on. If this mapping is wrong, every segment ends up in the wrong class and all downstream analysis is corrupted.

Rather than copying thousands of files and discovering a mistake afterward, this script walks through all segment folders, applies the mapping, and prints the count per category — without moving or copying a single file.

Setup & Usage

No new dependencies here. Once 03_segment_all_subjects.m has finished and you have the EEG_Segments/ folder on your Desktop, this script can be run as-is — it just reads the segment filenames already on disk and counts them.

Thought Process

The mapping comes directly from the BCI2000 dataset documentation and the task-motion_events.json file in 00_dataset/. Each subject completed three repetitions of each of four tasks, so runs 3, 7, and 11 are all Task 1 for every subject. The run number cycles through this pattern identically for all 109 subjects.

Running the count verification confirmed approximately 327 T1 files and 327 T2 files per task category (3 runs × 109 subjects). Some subjects have missing runs in the source dataset so the actual counts were slightly lower. Seeing those counts side by side before committing to the copy operation was the confirmation needed to proceed.

Class imbalance discovered here

The count also revealed that Eyes Open and Eyes Closed baselines would have only 109 files each (one per subject), while the movement task categories would have 2,000–18,000. This imbalance was not obvious until the counts were printed side by side and became a documented problem to address in the CNN training phase.
