Phase 2 — Analysis

Methods
& Signal Analysis

Fifteen scripts across MATLAB and Python covering signal exploration, frequency analysis, topographic brain mapping, and algorithm validation. Each page documents the thought process behind the script, the key code, and why each decision was made.

MATLAB — Signal Exploration
01
Time-Domain Exploration
First look at the cleaned signal: 1–50 Hz bandpass + 100ms moving average, three plots across all 64 channels.
› 02
FFT and Peak Frequency
High-pass at 8 Hz then FFT per channel. Normalized offset spectra + global peak frequency tracking.
› 03
Normalized Amplitude Spectra
Each channel normalized to max 1 and stacked with fixed offsets for readable cross-channel comparison.
› 04
Sliding-Window Dominant Frequency
1-second window, 100ms step. Extracts dominant frequency in 8–80 Hz over time per channel.
› 05
Batch Image Export — 3 Plots
First batch pipeline: saves original, filtered, and smoothed PNGs for every .set file in a folder.
› 06
Seven-Plot Analysis
Comprehensive single-file view: 7 plots covering time domain, FFT at 14–50 Hz, and averaged spectra.
› 07
Batch Image Export — 7 Plots
Scales the 7-plot analysis across an entire folder, producing a structured visual dataset.
› 08
Extract Average FFT Features
First ML-ready output: average FFT across 64 channels saved as compact .mat feature vectors.
›
Python — Analysis & Visualization
09
Artifact Detection
MNE-based quality check across 6 baseline files. Flags signal excursions beyond 3 std with Gaussian smoothing.
› 09b
Raw Data Plotting
Batch-loads all EDF files from a subject folder, converts the first recording to a PyTorch tensor, and opens MNE’s interactive signal browser.
› 10
EDF Viewer GUI
Full Tkinter desktop app with multi-file EDF import, channel browser, and embedded Python REPL with preloaded EEG tools.
› 11
Channel Position Mapper
Drag-and-drop GUI to place 64 channel labels on a head outline. Saves positions to CSV for topomap use.
› 12
PSD Comparison Across Categories
Welch PSD for all 11 movement categories. Percentage-match algorithm filters outliers before averaging.
› 13
Topomap Prototype
Proof-of-concept topomap with placeholder data. Validates cubic interpolation + head masking geometry.
› 14
Full Frequency-Band Topomaps
Complete pipeline: per-channel band power (delta–gamma) interpolated onto head grid for all 11 conditions.
› 15
Pattern Match Validation
Validates the 15%-tolerance percentage-match algorithm on 25 synthetic noisy datasets before trusting it on real EEG.
› 16
EEG Channel Statistics
Loads a single EDF file and prints per-channel mean, min, and max amplitude in µV. Diagnostic baseline for the simulated EEG in Script 17.
› 17
Simulated EEG
Generates 64-channel synthetic EEG using per-channel amplitude statistics from a real subject. Momentum random walk constrained to measured bounds.
› 18
Multi-File EEG Plot
Loads a named list of segmented EDF trials and renders one stacked time-domain plot per file for direct visual comparison across segments.
› 19
DTW Pattern Match
Pairwise cross-correlation across 10 baseline EEG files to identify consistent signal patterns. Flags files that fail to match at least half the group.
› 20
DTW Variant Match
Extends Script 19 with true Dynamic Time Warping and a synthetic signal-variant generator for more permissive outlier detection on time-shifted recordings.
