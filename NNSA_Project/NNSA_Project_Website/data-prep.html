Phase 1 — MATLAB

Data Preparation
Pipeline

Seven scripts that take raw continuous EDF recordings from 109 subjects and produce organized, labeled segments ready for signal analysis and CNN training. Each script is documented with its full thought process and key code on its own page.

Segmentation
01
Single-File Segmentation
Proof-of-concept: parse one event file, slice one EDF into labeled segments. Validates the regex and sample math before scaling up.
› 02
Folder-Level Segmentation
Wraps the single-file logic in a loop over all EDFs in a directory, creating a named output subfolder per file.
› 03
All-Subjects Batch Processing
Full batch pipeline iterating over all 109 subject folders, depositing every segment into a master EEG_Segments folder.
› 04
Category Count Verification
Dry-run that counts how many segments would land in each of the 9 classes using the run-number-to-task mapping — no files moved.
› 05
Sort into Classification Folders
Actually copies segments into EEG_Classification with 9 named subfolders. Reveals the baseline class-imbalance problem.
› 06
Convert .set Files to EDF
Converts the classified EEGLAB .set files back to .edf format for compatibility with Python's MNE library.
› 07
Generate EEGLAB Scroll Plots
Uses EEGLAB's pop_eegplot to render and save a scroll-view PNG for every file — a manual visual quality check.
