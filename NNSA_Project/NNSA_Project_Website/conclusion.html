10 Run Evaluation Results and Reflections

What the Data Actually Shows

Each CNN script was run 10 independent times with identical hyperparameters and no fixed random seed. The results below replace any single run numbers from earlier in the project and represent the honest picture of what these architectures can and cannot do on this dataset.

What Is Actually Working
Confirmed Results
  • Binary CNN Scripts 01 and 02 70.2% and 78.4% mean both are 20 plus points above the 50% chance baseline and consistent across runs
  • Full segmentation and labeling pipeline runs end to end on all 109 subjects
  • FFT feature extraction MATLAB to mat to Python pipeline works correctly
  • Topographic frequency band maps generate correctly and show interpretable alpha lateralization
  • PSD comparison shows Eyes Closed and Rest separating cleanly in 8 to 12 Hz band
  • All processed data publicly accessible on Zenodo
Confirmed Problems
  • 11 class raw EDF 17.6% mean near 9.1% chance shallow 1D CNN cannot learn motor class distinctions from raw time series
  • 11 class FFT at 50 epochs 20.5% mean the architecture is systematically undertrained the earlier 60.23% single run was at 100 epochs and is plausibly real but unverified
  • High initialization sensitivity in binary scripts 29 point spread 50 to 79.6% without a fixed seed
  • Spectrogram early stopping is unstable best checkpoint captured at epoch 1 in some runs measuring initial weights not learned features
  • No artifact rejection applied before CNN training
  • No fixed random seed results not reproducible run to run
Priority Next Steps
  • Fix random seed in all CNN scripts to make results reproducible
  • Retest Script 04 at 150 epochs the 3 point spread across 10 runs at 50 epochs means every run hits the same local minimum before convergence more training time is required
  • Add proper test split to spectrogram CNN so reported accuracy is on held out data not validation checkpoint
  • Apply bandpass filtering and artifact threshold rejection before CNN input
  • Investigate EEGNet or ShallowConvNet for the 11 class problem purpose built EEG architectures with depthwise separable convolutions are more appropriate
  • Explore per band power delta theta alpha beta as CNN input rather than raw time series or averaged FFT
Script by Script Analysis

Scripts 01 and 02 Binary Classification The Confirmed Results

Both binary scripts classify Eyes Open vs Eyes Closed and both work. Script 01 raw 1D CNN reaches 70.2% mean Script 02 2D spectrogram CNN reaches 78.4% mean validation accuracy. The physiological reason is straightforward Eyes Closed suppresses posterior alpha activity 8 to 13 Hz a strong and well documented signal that even a shallow CNN detects reliably. The 8 point gap suggests that frequency structure from spectrogram conversion is helpful though comparison is imperfect since Script 02 reports validation accuracy not held out test accuracy. The dominant issue is initialization sensitivity without a fixed seed binary accuracy collapses to chance in bad runs. Fixing seed and using proper splits would make results fully rigorous.

Scripts 03 and 04 11 Class Classification What These Results Mean

Both 11 class scripts run the same motor categories Rest Eyes Open Eyes Closed Real Left Right Fist Imagined Left Right Fist Real Imagined Both Fists Real Imagined Both Feet. Script 03 uses raw EDF time series Script 04 uses average FFT features. Both reach 17 to 20% mean accuracy vs 9.1% chance a small but real improvement above baseline.

The difference lies in failure mode Script 03 is architecturally insufficient a shallow 1D CNN cannot learn frequency selective filters for motor imagery. Script 04 is plausible but undertrained the 3.21 point spread across runs 19.3 to 22.5% shows convergence to the same local minimum. The earlier 60.23% result at 100 epochs suggests training length is the limiting factor. 150 plus epochs is the next logical test.

The Most Important Correction From 10 Run Testing

Earlier claims that Script 04 was best were based on a single run at 100 epochs while all 10 run experiments used 50 epochs. The 20.5% mean does not contradict the earlier result it shows 50 epochs is insufficient for convergence. The correct next experiment is 10 runs at 150 epochs with a fixed seed. If accuracy rises near 60% FFT features are valid. If it remains near 30% the limitation is the representation itself.

Broader Context

Motor imagery EEG classification on BCI2000 is a standard benchmark. Published results for similar setups reach 60 to 80% with subject specific calibration artifact rejection and specialized architectures. The binary results here already fall within expected ranges for simpler tasks. The gap in the 11 class case is not a data limitation but an architecture and training limitation.

If 11 class accuracy can be pushed into the 60 to 70% range reliably it would enable meaningful applications in VR based motor control and neuroadaptive systems. The current pipeline from BCI2000 through MATLAB preprocessing to PyTorch CNN already demonstrates usable signal extraction. The remaining work is model and training optimization.
