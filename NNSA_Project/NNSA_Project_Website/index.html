Decoding Motor Imagery
from EEG Signals

This project builds a CNN based classifier to distinguish whether a person is performing or imagining a physical movement, using a 64 channel scalp EEG dataset from 109 subjects. Work proceeded in two phases: a MATLAB pipeline to segment and organize raw recordings, followed by Python based spectral analysis, topographic brain mapping, and neural network experimentation.

What This Project Does

EEG motor imagery classification is the problem of reading a brain signal and inferring what movement a person is thinking about — without them moving at all. This has direct applications in brain machine interfaces, VR kinesiology simulation, and assistive robotics. The open source BCI2000 dataset from Schalk et al. (2004) provides recordings of 109 subjects performing and imagining hand and foot movements, giving a large enough sample to attempt a data driven neural network approach.

The pipeline starts at raw continuous EDF recordings and ends at labeled, classified segments ready for CNN training. Along the way, every intermediate decision — how to parse event files, how to map run numbers to task types, what frequency range to filter, how to structure the CNN input — is documented here with the actual code that made each choice.

How to Navigate

Why Are There 14 Event Classes but Only 11 CNN Classes?

The original PhysioNet BCI2000 dataset contains 14 unique event markers. These include two baseline recordings (eyes open and eyes closed) and twelve task related events distributed across four motor tasks. Each task contains a rest period (TASKxT0) and two movement cues (TASKxT1 and TASKxT2).

Although the dataset contains 14 event markers, the CNN was not trained to predict all 14 independently. The four task specific rest markers (TASK1T0, TASK2T0, TASK3T0, and TASK4T0) all represent the same underlying brain state: no movement instruction. During preprocessing, these four events were merged into a single Rest category.

This reduced the classification problem from 14 dataset events to 11 CNN classes: one Rest class plus ten movement or motor imagery classes. The baseline eyes open and eyes closed recordings were analyzed separately as a binary benchmark experiment and were not included in the final 11 class motor imagery model.
