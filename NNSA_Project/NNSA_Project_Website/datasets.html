Zenodo and Data Access

Processed Datasets

The raw BCI2000 source data lives on OpenNeuro. All processed derivatives including segmented signals, classified folders, EDF conversions, and CNN feature vectors are hosted on Zenodo in two records, linked below. The data lineage diagram shows exactly how each dataset was produced from the one above it.

Data Lineage

Source: OpenNeuro ds004362
Raw continuous EDF recordings plus event files for 109 subjects, 14 runs each. BCI2000 and Schalk et al. 2004. OpenNeuro DOI
EEG Segments Set Format Zenodo Record 1
03 segment all subjects.m slices every EDF at T0 T1 T2 event boundaries and saves as set and fdt. One subfolder per run.
EEG Classification Set Format Zenodo Record 1
05 sort into class folders.m reorganizes the same segments into 9 class subfolders (Rest, Task1 Real Left Fist, etc.) using the run number to task mapping.
EEG Classification EDF Format Zenodo Record 1
06 convert set to edf.m converts every set to edf format, preserving the 9 class folder structure. Enables use with Python MNE.
EEG CNN Dataset EDF Zenodo Record 1
11 folder EDF dataset including Baseline Eyes Open and Baseline Eyes Closed added from source separately. This is the primary CNN training input.
EEG CNN Features Avg FFT Zenodo Record 2
08 extract avg fft features.m applies 14 to 50 Hz bandpass plus moving average then averages FFT across all 64 channels. One avg fft mat per segment containing avg fft row and f vectors.

Zenodo Record 1 Processed Segment Data

EEG Motor Imagery Processed Segment Data BCI2000 109 Subjects

Contains all four intermediate and final processed datasets in their full form. Intended for researchers who want to replicate the preprocessing pipeline or run their own analysis on the classified segments.

EEG Segments Set Format
All T0 T1 T2 event segments from all 109 subjects in EEGLAB set fdt format. Organized by run not by class. Intermediate output of the segmentation pipeline.
Format: set fdtSubjects: 109Script: 03 segment all subjects.m
EEG Classification Set Format
Same segments as above reorganized into 9 class subfolders by run number to task mapping. The set fdt version of the classified dataset.
Format: set fdtClasses: 9Script: 05 sort into class folders.m
EEG Classification EDF Format
EEG Classification Set Format converted to EDF. Same 9 class folder structure, compatible with Python MNE. Used to build EEG CNN Dataset EDF.
Format: edfClasses: 9Script: 06 convert set to edf.m
EEG CNN Dataset EDF
Primary CNN training dataset. 11 class folders including Baseline Eyes Open and Baseline Eyes Closed sourced directly from raw recordings. Best quality segmented EDF files.
Format: edfClasses: 11Used by: p.py p4.py

EEG Motor Imagery CNN Feature Vectors Average FFT BCI2000 109 Subjects

Compact ML ready feature vectors. Intended for researchers who want to train a classifier directly without reprocessing the raw EEG. Much smaller download than Record 1.

EEG CNN Features Avg FFT
One avg fft mat file per segment. Each file contains avg fft row average FFT across all 64 channels after 14 to 50 Hz bandpass and 100ms smoothing and f frequency vector. Directly loadable into a CNN or feature matrix.
