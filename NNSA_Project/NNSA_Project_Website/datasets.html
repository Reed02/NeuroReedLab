Zenodo & Data Access

Processed Datasets

The raw BCI2000 source data lives on OpenNeuro. All processed derivatives — segmented signals, classified folders, EDF conversions, and CNN feature vectors — are hosted on Zenodo in two records, linked below. The data lineage diagram shows exactly how each dataset was produced from the one above it.

Data Lineage

Source: OpenNeuro ds004362
Raw continuous EDF recordings + event files for 109 subjects, 14 runs each. BCI2000 / Schalk et al. 2004. OpenNeuro DOI
EEG_Segments_SetFormat — Zenodo Record 1
03_segment_all_subjects.m slices every EDF at T0/T1/T2 event boundaries and saves as .set + .fdt. One subfolder per run.
EEG_Classification_SetFormat — Zenodo Record 1
05_sort_into_class_folders.m reorganizes the same segments into 9 class subfolders (Rest, Task1_Real_Left_Fist, etc.) using the run-number-to-task mapping.
EEG_Classification_EDFFormat — Zenodo Record 1
06_convert_set_to_edf.m converts every .set to .edf format, preserving the 9-class folder structure. Enables use with Python MNE.
EEG_CNN_Dataset_EDF — Zenodo Record 1
11-folder EDF dataset including Baseline_Eyes_Open and Baseline_Eyes_Closed (added from source separately). This is the primary CNN training input.
EEG_CNN_Features_AvgFFT — Zenodo Record 2
08_extract_avg_fft_features.m applies 14–50 Hz bandpass + moving average then averages FFT across all 64 channels. One _avgFFT.mat per segment containing avg_fft_row + f vectors.

Zenodo Record 1 — Processed Segment Data

EEG Motor Imagery — Processed Segment Data (BCI2000, 109 Subjects)

Contains all four intermediate and final processed datasets in their full form. Intended for researchers who want to replicate the preprocessing pipeline or run their own analysis on the classified segments.

Open on Zenodo ›
EEG_Segments_SetFormat
All T0/T1/T2 event segments from all 109 subjects in EEGLAB .set/.fdt format. Organized by run, not by class. Intermediate output of the segmentation pipeline.
Format: .set / .fdtSubjects: 109Script: 03_segment_all_subjects.m
EEG_Classification_SetFormat
Same segments as above, reorganized into 9 class subfolders by run-number-to-task mapping. The .set/.fdt version of the classified dataset.
Format: .set / .fdtClasses: 9Script: 05_sort_into_class_folders.m
EEG_Classification_EDFFormat
EEG_Classification_SetFormat converted to EDF. Same 9-class folder structure, compatible with Python MNE. Used to build EEG_CNN_Dataset_EDF.
Format: .edfClasses: 9Script: 06_convert_set_to_edf.m
EEG_CNN_Dataset_EDF
The primary CNN training dataset. 11 class folders including Baseline_Eyes_Open and Baseline_Eyes_Closed sourced directly from the raw recordings. Best-quality segmented EDF files.
Format: .edfClasses: 11Used by: p.py, p4.py

Zenodo Record 2 — CNN Feature Vectors

EEG Motor Imagery — CNN Feature Vectors, Average FFT (BCI2000, 109 Subjects)

Compact ML-ready feature vectors. Intended for researchers who want to train a classifier directly without reprocessing the raw EEG. Much smaller download than Record 1.

Open on Zenodo ›
EEG_CNN_Features_AvgFFT
One _avgFFT.mat file per segment. Each file contains avg_fft_row (average FFT across all 64 channels after 14–50 Hz bandpass and 100ms smoothing) and f (frequency vector). Directly loadable into a CNN or feature matrix.
Format: .matContents: avg_fft_row + fScript: 08_extract_avg_fft_features.m
