This Project Is a Continuation and Enhancement Upon my NNSA Open Source EEG Classificaion Project

GOAL:
The goal of this project is to use a muse2 eeg headset to accurately classify finger movements and map them to a Hiwonder uHand.
Upon completion augmentation of the uHand to attach it to a robotic arm for arm classifcation and enhancement of hand movement
will be attempted.
